About

This program was created by CodeMasters360 for teaching children how to read clocks in English.

  • Design & Development: CodeMasters360 Team
  • Audio: Child-friendly, clear English recordings
  • Contact: codemasters360@gmail.com

How to Use

  1. Enter your name when prompted. Your progress will be saved.
  2. Use Learn Time to practice reading clocks and hearing the correct pronunciation.
  3. Try the Quiz modes:
    • Text Choice: Read the clock and select the correct time from the options.
    • Audio to Text: Listen to the time and select the correct text answer.
    • Audio to Clock: Listen to the time and select the correct clock image.
  4. View your daily progress and scores in the Scores section.
